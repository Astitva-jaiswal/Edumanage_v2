# Edumanage_v2
Production-grade Education Management System built using Spring Boot, PostgreSQL, JWT Authentication, Docker and Swagger.

## Features

### Admin
- Manage Teachers
- Manage Courses
- Manage Notices

### Teacher
- Mark Attendance
- Create Assignments
- Create Quizzes

### Student
- View Attendance
- Submit Assignments
- View Notices

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Spring Security
- JWT
- Swagger
- Docker
- JUnit
- Mockito
